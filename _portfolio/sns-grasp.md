<span class="project-name">SNS-Grasp</span> is an intent-driven grasp generation framework that uses semantic
information to guide robotic grasping. It explores how semantic signals can help
robotic systems better understand the intended use of objects and produce more
context-aware grasping behaviors.

This work was accepted to <strong>AAAI 2026</strong>.

## Research Question

How can semantic information help robotic systems infer intent and generate more
human-aligned grasps?
